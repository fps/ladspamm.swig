# ladspamm.swig

SWIG generated wrappers for ladspamm

Note that right now only python bindings will be generated. It should be easy to adopt the makefile to create bindings for different languages though.

# Building

Type 

<pre>
make
</pre>

# Installation

Type 

<pre>
make install
</pre>
